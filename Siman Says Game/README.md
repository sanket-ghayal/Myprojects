Simon Says Game
A simple Simon Says memory game built with JavaScript, HTML, and CSS.
The goal is to remember and repeat the sequence of colors.


How to Play
1. Press the Start button (or any key, depending on version).
2. Watch the sequence of colors blink.
3. Repeat the sequence by clicking the buttons in the correct order.
4. Each round adds a new color to the sequence.
5. If you tap the wrong color, the screen blinks red, and the game restarts.


Tech Stack
HTML – structure
CSS – styling and animations
JavaScript – game logic
